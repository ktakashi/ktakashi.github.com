Reader macros for SRFIs

SRFI-4

The SRFI-4 also defines its reader macro. Sagittarius also suppots these. It defines tagged vector and the tags can be s8, u8, s16, u16, s32, u32, s64, u64, f32 or f64. For each value of tags, the external representation of instances of the vector is #_tag_(... elements ...)On Sagittarius, these reader macros are not automatically enabled. You need to explicitly import it. For more detail, see (sagittarius reader) - reader macro library.
