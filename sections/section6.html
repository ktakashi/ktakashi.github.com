Sagittarius extensions

Sagittarius has its own extension libraries because even R6RS is huge however I know it is not sufficient to write practical program. To support to write it, Sagittarius provides some useful libraries.

  1. (sagittarius) - builtin library
  2. (sagittarius control) - control library
  3. (sagittarius ffi) - Foreign Function Interface
  4. (sagittarius filewatch) - Monitoring filesystem
  5. (sagittarius io) - Extra IO library
  6. Sagittarius MOP
  7. (sagittarius object) - Convenient refs and coercion procedures
  8. (sagittarius process) - Process library
  9. (sagittarius reader) - reader macro library
  10. (sagittarius record) - Extra record inspection library
  11. (sagittarius regex) - regular expression library
  12. (sagittarius socket) - socket library
  13. (sagittarius threads) - Thread
  14. (sagittarius timezone) - Timezone
  15. (sagittarius debug) - Debugging support
  16. (sagittarius generators) - Generators
  17. Cryptographic libraries
