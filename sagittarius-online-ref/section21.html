2.1Invoking sash

Sagittarius can be used either as an independent Schame interpreter or an embedded Scheme library. The interpreter which comes with Sagittarius destribution is a program named sash.

Program sash [options] scheme-file arg ...
Invoking sash. If scheme-file is not given, it runs with interactive mode. Detail options are given with option "-h".
