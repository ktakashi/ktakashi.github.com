Sagittarius Scheme

Sagittarius Scheme is a R6RS1/R7RS2 Scheme implementation with a lot of practical libraries, especially cryptographic libraries.

  1. The Revised^6 Report on the Algorithmic Language Scheme

  1. The Revised^7 Report on the Algorithmic Language Scheme

You can find the source archive and/or Windows binary from the main repository.
